SearchableMedia
===============

- Compile & Run with Android Studio

- After install to ATV, try following command on host
  adb shell am start -a android.intent.action.ASSIST --es query \"Big Buck Bunny\"

- It is a simplified version of Assistant on Android-TV of the link below :
  https://github.com/googlearchive/leanback-assistant

- Posting about the original sample source :
  https://android-developers.googleblog.com/2017/10/video-playback-with-google-assistant-on.html
